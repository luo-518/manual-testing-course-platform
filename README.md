# 手工测试项目 – 在线课程报名系统

本项目为一个完整的手工测试练习，展示了从需求分析、测试用例设计、测试执行到缺陷报告输出的完整测试流程。

---

##  项目简介

**在线课程报名系统**是一个模拟的 Web 平台，支持用户：

- 浏览课程
- 关键词搜索课程
- 查看课程详情
- 加入报名清单
- 提交报名订单
- 完成模拟支付
- 查看历史报名记录

本仓库包含围绕该系统编写的完整手工测试文档。

## 本项目测试范围

本次测试覆盖：

- 功能测试（Function Testing）
- 黑盒测试（Black-box testing）
- 需求驱动测试（Requirement-based Testing）
- 等价类划分（Equivalence Partitioning）
- 边界值分析（Boundary Value Analysis）
- 场景法（Scenario-based Testing）
- 错误推测法（Error Guessing）
